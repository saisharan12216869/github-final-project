This repository contains the final project for the course

## Simple Interest Calculator

This project includes a Bash script that calculates **simple interest**
based on user input.

### Input
- Principal amount (p)
- Rate of interest per year (r)
- Time period in years (t)

### Output
- Simple Interest = (p × r × t) / 100

The script prompts the user to enter values and displays the calculated
simple interest.
